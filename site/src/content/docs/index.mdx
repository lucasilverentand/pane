## Built for Real Dev Sessions

`pane` is a Rust terminal multiplexer that keeps state in a local daemon while
lightweight TUI clients attach and detach as needed. It is aimed at the kind of
workflow where shells, editors, AI agents, and local servers all need to stay
alive without turning one terminal tab into your entire operating environment.

## Quick Start

```sh
cargo install --path crates/pane-tui
pane
```

Once attached:

- `i` or `Enter` enters Interact mode
- `Ctrl+Space` returns to Normal mode
- `:` opens the command palette
- `n` opens the tab picker
- `o` toggles the workspace overview
